# 📈 AI Trading Analyst

> **데이터 기반**의 분석과 **AI**를 결합한 **주식 지표 분석 플랫폼**

**Top-Down 방식**을 웹 인터페이스로 구현하여,  
복잡한 차트 없이도 **AI와 알고리즘이 분석한 전략과 점수**를 한눈에 확인할 수 있습니다.

<br>

## ✨ 주요 기능 (Key Features)

### 1️⃣ 🧠 AI Advisory
> * Google Gemini 2.0 Flash 모델 연동  
> * 실시간 차트 데이터를 기반으로 전문가 수준의 코멘트 제공  
> * 수급(외인/기관) 데이터와 애널리스트 평균 목표가를 종합하여 분석
> * API Key를 사용자의 **Local Storage에서만 관리**하여 안전하게 사용 가능

<br>

### 2️⃣ 🎯 멀티 타임프레임 & ATR 전략
| 전략타입 | 기준 데이터 | 활용 목적 |
|----------|-------------|-----------|
| 단타 | 60분봉 데이터 | 단기 진입 타점 |
| 스윙 | 일봉 데이터 | 안정적인 추세 추종 |
| 장투 | 주봉 데이터 | 중장기 흐름 판단 |

**ATR** 기반으로 **목표가(TP)**와 **손절가(SL)**를 자동 산출합니다.

<br>

### 3️⃣ 📊 Top-Down 분석 UI 구성

| 구역 | 설명 |
|------|------|
| Market & Trend | VIX, 회전율, 이동평균선 등 시장의 큰 흐름 판단 |
| Trading Timing | RSI, 볼린저밴드, 스토캐스틱 등 매수/매도 타점 점수화 |

<br>

### 4️⃣ 🔐 BYOK (Bring Your Own Key) 아키텍처
- **보안 강화**: 서버에 API Key 저장 ❌  
- **LocalStorage**에만 저장하여 해킹 위험 최소화  
- **KIS API** 연동 시 실시간 시세 및 투자자별(개인/외인/기관) 당일 수급 분석 가능

## 📊 지표 설명 및 점수 산출 로직 (Indicators & Scoring Logic)
> 사용자 설정 **가중치(Weight)** 기반  
> **0 ~ 100점 스코어링 (Weighted Average 방식)**

<br>

### 🟣 Section 1: Market & Trend

#### 📌 이동평균선 (MA)
| 조건 | 점수 | 의미 |
|------|------|------|
| 이격도 95~103% (20일선 근처) | ⭐ 90점 | 눌림목지지 |
| 골든크로스 (5일선 > 20일선) | +가산점 | 추세 전환 초기 |
| 이격도 ≥ 110% | 20점 (감점) | 단기 과열 (추격 매수 위험) |
| 이격도 ≤ 90% | 40점 | 역배열 하락세 심화 |

#### 📌 MACD
| 상태 | 점수 |
|------|------|
| 0선 아래 + 골든크로스 | ⭐ 100점 | 바닥권 추세 전환 (강력 매수) |
| 0선 아래 + 상승 지속 | 80점 | 바닥 탈출 시도 |
| 0선 위 + 골든크로스 | 60점 | 이미 상승 중 (가격 부담 있음) |
| 데드크로스 | 0점 | 하락 반전 위험 |

#### 📌 수급 (Investors)
- 점수화되지 않지만 **개인/외국인/기관의 당일 순매수량**을 실시간으로 표기
- 주가는 하락하는데 **외인/기관**이 매수 중이라면 긍정적 시그널(매집)로 판단

#### 📌 OBV  
- 점수에는 반영되지 않으나 **세력 매집 판단용 참고 지표**
- 주가 하락 중 → OBV 상승/유지 = 긍정적 신호

#### 📌 VIX (공포지수)
| 수치 | 의미 |
|------|------|
| ≥ 20 | 공포 (변동성 확대 → 저점 매수 기회) |
| ≤ 15 | 평온 (안정적 상승장) |

#### 📌 회전율 (Turnover)
| 회전율 | 의미 |
|--------|------|
| ≥ 5% | 급등 관심 수급 집중 |
| ≤ 1% | 소외 종목 |


### 🟢 Section 2: Trading Timing (매수/매도 타이밍 지표)

#### 📌 RSI
> 점수 공식: **100 - RSI 값 (낮을수록 고득점)**

| RSI 값 | 해석 |
|--------|------|
| ≤ 30 | 과매도 → 매수 기회 |
| ≥ 70 | 과매수 → 매도 신호 |


#### 📌 스토캐스틱 (Stochastic) - 다이버전스
> 점수 공식: **100 - K값 + 시그널 가산점**

| K 값 | 의미 |
|------|------|
| 과매도(20↓) + 골든크로스 | ⭐ 100점 (강력 매수) |
| 상승 다이버전스 | +20점 가산 (주가 하락, 지표 상승) |
| 과매수(80↑) + 데드크로스 | 0점 (매도 시그널) |


#### 📌 볼린저 밴드 (Bollinger Band)
> 공식: **(1 - %B) × 100**

| %B 위치 | 점수 | 의미 |
|---------|------|------|
| < 0.2 (하단) | ⭐ 80~100 | 과매도 구간 → 밴드 회귀 본능 이용 |
| > 0.9 (상단) | 0~10 | 과매수 구간 → 저항 가능성 |



#### 📌 ATR 전략 (TP/SL 자동 계산)
| 구분 | ATR 활용 방식 |
|------|---------------|
| 단타 | 60분봉 ATR |
| 스윙 | 일봉 ATR × 2 |
| 장투 | 주봉 ATR × 3 |
