# 📈 AI Trading Analyst

> **데이터 기반**의 분석과 **AI**를 결합한 **주식 투자 보조 플랫폼**

**Top-Down 방식**을 웹 인터페이스로 구현하여,  
복잡한 차트 없이도 **AI와 알고리즘이 분석한 전략과 점수**를 한눈에 확인할 수 있습니다.

<br><br>

## ✨ 주요 기능 (Key Features)

### 1️⃣ 🧠 AI Advisory
> Google Gemini 2.0 Flash 모델 연동  
> 실시간 차트 데이터를 기반으로 전문가 수준의 코멘트 제공  
> API Key를 사용자의 **Local Storage에서만 관리**하여 안전하게 사용 가능

<br>

### 2️⃣ 🎯 멀티 타임프레임 & ATR 전략
| 전략타입 | 기준 데이터 | 활용 목적 |
|----------|-------------|-----------|
| 단타 | 60분봉 데이터 | 단기 진입 타점 |
| 스윙 | 일봉 데이터 | 안정적인 추세 추종 |
| 장투 | 주봉 데이터 | 중장기 흐름 판단 |

**ATR** 기반으로 **목표가(TP)**와 **손절가(SL)**를 자동 산출합니다.

<br>

### 3️⃣ 📊 Top-Down 분석 UI 구성

| 구역 | 설명 |
|------|------|
| Market & Trend | VIX, 회전율, 이동평균선 등 시장의 큰 흐름 판단 |
| Trading Timing | RSI, 볼린저밴드, 스토캐스틱 등 매수/매도 타점 점수화 |

<br>

### 4️⃣ 🔐 BYOK (Bring Your Own Key) 아키텍처
- **보안 강화**: 서버에 API Key 저장 ❌  
- **LocalStorage**에만 저장하여 해킹 위험 최소화  
- **KIS API** 연동 시 실시간 시세 분석 가능 (미입력 시 20분 지연 시세 사용)

## 📊 지표 설명 및 점수 산출 로직 (Indicators & Scoring Logic)
> 사용자 설정 **가중치(Weight)** 기반  
> **0 ~ 100점 스코어링 (Weighted Average 방식)**

<br>

### 🟣 Section 1: Market & Trend

#### 📌 이동평균선 (MA)
| 조건 | 점수 |
|------|------|
| 정배열 (5 > 20 > 60 > 120) | ⭐ 100점 |
| 단기 정배열 (5 > 20) | 70점 |
| 역배열 | 🚨 0점 |
| 골든크로스 (5vs20, 20vs60) | +20~30점 가산 |

#### 📌 MACD
| 상태 | 점수 |
|------|------|
| 상승 가속 (0선 위) | 100점 |
| 반등 시도 (0선 아래) | 75점 |
| 하락 가속 | 0점 |

#### 📌 OBV  
- 점수에는 반영되지 않으나 **세력 매집 판단용 참고 지표**
- 주가 하락 중 → OBV 상승/유지 = 긍정적 신호

#### 📌 VIX (공포지수)
| 수치 | 의미 |
|------|------|
| ≥ 20 | 공포 (변동성 확대 → 저점 매수 기회) |
| ≤ 15 | 평온 (안정적 상승장) |

#### 📌 회전율 (Turnover)
| 회전율 | 의미 |
|--------|------|
| ≥ 5% | 급등 관심 수급 집중 |
| ≤ 1% | 소외 종목 |


### 🟢 Section 2: Trading Timing (매수/매도 타이밍 지표)

#### 📌 RSI
> 점수 공식: **100 - RSI 값 (낮을수록 고득점)**

| RSI 값 | 해석 |
|--------|------|
| ≤ 30 | 과매도 → 매수 기회 |
| ≥ 70 | 과매수 → 매도 신호 |


#### 📌 스토캐스틱 (Stochastic)
> 점수 공식: **100 - K값 (민감한 단기 타이밍 포착)**

| K 값 | 의미 |
|------|------|
| < 20 | 바닥권 (매수 타점) |
| > 80 | 고점 과열 (매도 타이밍) |


#### 📌 볼린저 밴드 (Bollinger Band)
> 공식: **(1 - %B) × 100**

| %B 위치 | 점수 | 의미 |
|---------|------|------|
| < 0 | ⭐ 80~100 | 하단 터치 → 반등 가능성 |
| > 1 | ⚠️ 0~20 | 상단 돌파 → 단기 고점 |



#### 📌 ATR 전략 (TP/SL 자동 계산)
| 구분 | ATR 활용 방식 |
|------|---------------|
| 단타 | 60분봉 ATR |
| 스윙 | 일봉 ATR × 2 |
| 장투 | 주봉 ATR × 3 |

## ⚙️ 설정 가이드 (Settings)

| 설정 항목 | 설명 |
|-----------|------|
| Gemini API Key | AI 리포트 생성용. Google AI Studio에서 무료 발급 |
| KIS API | 국내 실시간 시세 분석 가능 (선택 입력) |
| 지표 가중치 설정 | 투자 스타일에 따른 커스터마이징 |

💡 예시  
> 추세매매 스타일 → MA, MACD 가중치 UP  
> 단타 스캘핑 → RSI, Stochastic 가중치 UP
