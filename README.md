# 📈 AI Trading Analyst (AI 주식 분석기)

> **데이터 기반의 기술적 분석(Technical Analysis)과 생성형 AI(Gemini)의 통찰력을 결합한 하이브리드 주식 투자 보조 플랫폼**

이 프로젝트는 주식 투자의 정석인 **Top-Down 방식(시장/추세 파악 → 매매 타이밍 결정)**을 웹 인터페이스로 구현했습니다. 사용자는 복잡한 차트 설정 없이도 AI와 알고리즘이 분석한 **매매 점수**와 **전략**을 한눈에 확인할 수 있습니다.

---

## ✨ 주요 기능 (Key Features)

### 1. 🧠 AI 투자 자문 (AI Advisory)
- **Google Gemini 2.0 Flash** 모델을 연동하여 현재 차트 데이터를 기반으로 전문가 수준의 코멘트를 제공합니다.
- 사용자의 **개인 API Key**를 사용하여 비용 부담 없이 무제한 분석이 가능합니다.

### 2. 🎯 멀티 타임프레임 & ATR 전략
- **단타(60분), 스윙(일봉), 장투(주봉)** 관점을 통합하여 분석합니다.
- 변동성 지표인 **ATR(Average True Range)**을 기반으로 과학적인 **목표가(TP)**와 **손절가(SL)**를 자동으로 산출합니다.

### 3. 📊 Top-Down 분석 UI
- **Market & Trend:** VIX(공포지수), 회전율(수급), 이동평균선 등 시장의 큰 흐름을 먼저 보여줍니다.
- **Trading Timing:** RSI, 볼린저밴드, 스토캐스틱 등 구체적인 진입 시점을 점수화하여 보여줍니다.

### 4. 🔐 BYOK (Bring Your Own Key) 아키텍처
- **보안:** 서버에 개인 키를 저장하지 않고, 사용자의 브라우저(Local Storage)에만 저장하여 안전합니다.
- **실시간 시세:** 한국투자증권 API 키를 입력하면 지연 없는 실시간 시세 분석이 가능합니다. (미입력 시 20분 지연 시세 사용)

## 📊 지표 분석 로직 (Indicator Logic)

분석 화면은 크게 두 가지 섹션으로 나뉩니다.

### 🟣 Section 1: Market & Trend (숲을 보는 단계)
*시장의 방향성과 종목의 대세 흐름을 파악하여 매매 여부를 1차적으로 결정합니다.*

| 지표 | 분석 내용 및 시그널 |
| :--- | :--- |
| **VIX (공포지수)** | • **20 이상:** 공포 (변동성 확대, 저점 매수 기회)<br>• **15 이하:** 평온 (안정적 상승장) |
| **추세 (Trend)** | **주봉(장기) vs 일봉(단기)** 추세를 비교합니다.<br>• 🔥 **대세 상승:** 주봉↑ 일봉↑ (적극 매수)<br>• 🌊 **눌림목:** 주봉↑ 일봉↓ (가장 안전한 매수 기회)<br>• 🚧 **기술적 반등:** 주봉↓ 일봉↑ (단기 매매만 접근)<br>• ☁️ **대세 하락:** 주봉↓ 일봉↓ (관망) |
| **이동평균선 (MA)** | **5, 20, 60, 120일선**의 배열을 분석합니다.<br>• **정배열:** 100점 (강력 매수)<br>• **골든크로스:** 5vs20 또는 20vs60 돌파 시 가산점 |
| **MACD** | • **상승 가속:** 0선 위에서 시그널선 돌파<br>• **하락 둔화:** 0선 아래에서 반등 시도 |
| **회전율** | 전체 주식 수 대비 거래량을 봅니다. (5% 이상 시 폭발적 관심) |

### 🟢 Section 2: Trading Timing (나무를 베는 단계)
*구체적인 진입 가격과 매매 시점을 포착합니다.*

| 지표 | 분석 내용 및 시그널 |
| :--- | :--- |
| **종합 점수** | 모든 지표를 가중 평균하여 **0~100점**으로 환산합니다.<br>• **70점 이상:** 매수 우위<br>• **30점 이하:** 매도 우위 |
| **ATR 전략** | 변동성을 기반으로 손익비를 계산합니다.<br>• **단타:** 60분봉 ATR 기준 (타이트한 손절)<br>• **스윙:** 일봉 ATR × 2배 (표준 전략)<br>• **장투:** 주봉 ATR × 3배 (추세 추종) |
| **RSI** | • **30 이하:** 과매도 (기술적 반등 매수)<br>• **70 이상:** 과매수 (차익 실현) |
| **볼린저 밴드** | 주가의 상대적 위치(%B)를 파악합니다.<br>• **하단 터치:** 매수 기회 (높은 점수 부여)<br>• **상단 돌파:** 매도 기회 (낮은 점수 부여) |
| **스토캐스틱** | • **K < 20:** 침체권 (골든크로스 예상)<br>• **K > 80:** 과열권 |
